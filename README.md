## Project Overview
This project was built for my Dual Enrollment (DE) Web Page Development course. It is a multi-page digital magazine showcase styled after "Runway Magazine," featuring a modern review and a structured character lookbook for *The Devil Wears Prada 2*. The primary technical focus was on establishing clean multi-page site navigation and structural page flows.

## Key Technical Skills Demonstrated
* **Multi-Page Site Architecture:** Integrated independent HTML files (`index.html` and `lookbook.html`) utilizing relative linking paths to allow seamless user navigation across the platform.
* **Semantic Layout Structure:** Used semantic HTML elements (like headers, sections, and navigation menus) to ensure high accessibility and professional document outline practices.
* **CSS Typography & Design:** Structured modern digital magazine layouts utilizing crisp typography, grid structures, and interactive hover properties.
